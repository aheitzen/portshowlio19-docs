Randomization

To randomize content on your pages make sure you have the pink piece of code below in your conditional loops.

'orderby' => 'rand' 
$args = array(
   'post_type' => array('student'),
   'posts_per_page' => 100,
   'orderby'        => 'rand',
);
Additionally, in the WordPress dashboard go to the WP Engine tab and scroll down to the Advanced Configuration section. Make sure Allow ORDER BY RAND()dropdown is enabled.
